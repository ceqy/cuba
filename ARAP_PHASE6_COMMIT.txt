### Architecture Pattern

**Trait-based design:**
```rust
pub trait ExtendedArApMethods {
    async fn batch_get_partner_details(...) -> Result<...> {
        // Default implementation
        Ok(Response::new(...))
    }
}

impl ExtendedArApMethods for ArApServiceImpl {}

// Delegate from main trait
impl AccountsReceivablePayableService for ArApServiceImpl {
    async fn batch_get_partner_details(...) {
        <Self as ExtendedArApMethods>::batch_get_partner_details(self, request).await
    }
}
```

**Benefits:**
- Clean separation of concerns
- Easy to override specific methods
- Maintains single source of truth
- Supports incremental enhancement
