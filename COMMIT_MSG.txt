feat: standardize OpenAPI documentation generation workflow

## Summary
Unified OpenAPI documentation generation across all services (Auth, GL, AR/AP) 
by establishing a "Golden Template" standard. Eliminated fragmented scripts and 
inconsistent proto definitions.

## Key Changes

### 1. Unified Build Script
- Created `scripts/gen_openapi.sh` as single entry point for all services
- Replaced service-specific scripts: `gen_auth_openapi.sh`, `gen_finance_openapi.sh`, `gen_arap_openapi.sh`
- Automated Swagger 2.0 → OpenAPI 3.1 conversion pipeline

### 2. Self-Contained Proto Definitions
- Added top-level `openapiv2_swagger` options to all `.proto` files
- Embedded API metadata (title, version, security) directly in proto
- Eliminated dependency on external patching scripts (`add_tag_descriptions.py`)

**Updated files:**
- `protos/finance/gl/gl_journal_entry.proto`
- `protos/finance/ar_ap/ar_ap.proto`
- `protos/auth/auth_service.proto`

### 3. Standardized API Versioning
- Unified all service versions to `0.1.0`
- Documented versioning strategy in `docs/NEW_SERVICE_GUIDE.md`
- Clarified distinction between:
  - Swagger UI version (5.31.0)
  - OpenAPI spec version (3.1.0)
  - API service version (0.1.0)

### 4. Fixed Security Definitions
- Added Bearer Auth (`bearer_auth`) to all services
- Ensured consistent `securitySchemes` in generated OpenAPI 3.0 docs
- Fixed Auth service missing authorization definitions

### 5. Updated Swagger UI Configuration
- Corrected document paths in `scripts/start.sh` to match new directory structure
- Changed from flat paths (`/docs/finance/gl_journal_entry.openapi3.json`)
- To nested paths (`/docs/finance/gl/gl_journal_entry.openapi3.json`)

### 6. Documentation
- Created `docs/NEW_SERVICE_GUIDE.md` - comprehensive guide for creating new services
- Includes proto templates, versioning policy, and troubleshooting

### 7. Cleanup
- Removed obsolete scripts and backup files
- Updated `.gitignore` to exclude intermediate files (`*.swagger.json`)
- Organized docs into service-specific subdirectories
